# İhracat İstihbarat Ajanı 🌍

Uluslararası ticaret danışmanları için otomatik web sitesi analizi, 
GTİP tespiti, pazar araştırması ve rakip karşılaştırması yapan AI ajanı.

---

## 🚀 Kurulum (15 dakika)

### Adım 1 — GitHub Hesabı Aç
1. github.com → Sign Up → ücretsiz hesap oluştur

### Adım 2 — Repository Oluştur
1. github.com'a giriş yap
2. Sağ üst + butonu → "New repository"
3. Repository name: `ihracat-ajani`
4. Public seç
5. "Create repository" butonuna bas

### Adım 3 — Dosyaları Yükle
1. "uploading an existing file" linkine tıkla
2. Şu 2 dosyayı sürükle bırak:
   - `app.py`
   - `requirements.txt`
3. "Commit changes" butonuna bas

### Adım 4 — Streamlit'e Bağla
1. share.streamlit.io adresine git
2. "Sign in with GitHub" → giriş yap
3. "New app" butonuna bas
4. Repository: `ihracat-ajani` seç
5. Main file path: `app.py` yaz
6. "Deploy!" butonuna bas
7. 2-3 dakika bekle → linkini al!

### Adım 5 — API Key Ekle (GÜVENLİ)
1. Streamlit dashboard → app ayarları → "Secrets"
2. Şunu ekle:
```
ANTHROPIC_API_KEY = "sk-ant-xxxx"
```
3. Ya da kullanıcılar kendi key'lerini girebilir (uygulama bunu destekliyor)

## 📋 Özellikler

### Modül 1: Web Site Analizi (10 Madde)
- Ürün listesi ve teknik özellikler
- Sertifika kontrolü (CE, ISO, IEC vb.)
- Dil ve çeviri kalitesi
- SEO eksikleri
- Sosyal medya durumu
- Teknik doküman yeterliliği
- Garanti koşulları
- Teknik destek

### Modül 2: GTİP & Ürün Tespiti
- Otomatik GTİP kodu tespiti
- Teknik doküman analizi

### Modül 3: Pazar Analizi
- 3 hedef pazar önerisi ve gerekçeleri
- Pazar büyüklüğü ve büyüme oranı
- Türk üretici avantajları

### Modül 4: Vergi & Tarife
- İthalat vergi oranları
- KDV ve ek vergiler
- Türkiye anlaşmaları

### Modül 5: Sertifika & Belge
- Zorunlu sertifikalar
- Başvuru süreleri ve maliyetleri

### Modül 6: Lojistik
- Kara/deniz/hava karşılaştırması
- Navlun maliyetleri
- Incoterms önerisi

### Modül 7: Rakip Analizi
- 10 rakip otomatik tespiti
- Karşılaştırmalı analiz tablosu
- Güçlü/zayıf yönler
- Farklılaştırma stratejisi
